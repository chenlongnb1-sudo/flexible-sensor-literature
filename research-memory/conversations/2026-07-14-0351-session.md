## 当前研究方向

用户当前博士主线不是普通“高灵敏柔性压力传感器”，而是围绕 **柔性电子皮肤前端触觉计算** 展开：

- 结构-传感器-模拟前端-阵列-闭环协同设计。
- 低离散/容差触觉界面。
- ADC 前模拟矢量触觉读出。
- 低冗余矢量触觉阵列。
- 可编程物理触觉投影前端。
- 容错、可迁移、少样本校准的电子皮肤。
- 面向摩擦/纹理/滑移/三维力或方向相关触觉线索的柔性传感系统。

## 已有工作基础

### Paper A：低离散界面 + ADC 前模拟矢量触觉单元

核心身份：

> Analog vector tactile preprocessing before digitization for friction-aware soft electronic skins.

中文主张：

> 结构层降低输入离散性，模拟前端层把四象限响应压缩成方向相关矢量特征，服务纹理/摩擦识别和可靠决策。

已看到的关键证据：

- HCP/Hex 微结构与梯度叉指电极用于降低器件级响应离散性。
- G1-G4 对照中已有灵敏度和 CV 数据：G4 CV 可低至约 0.77%。
- 四象限压阻单元计划输出 Kz/Kx/Ky 或 Fz/Fx/Fy。
- 纹理识别/通道消融已有基础，vector/tangential 特征优于 z-only 或 reference force。

需要继续补强：

- raw A/B/C/D 与 hardware Kz/Kx/Ky 同步记录。
- hardware vector 与 software projection 一致性分析。
- Kx/Ky 方向相关标定。
- PSD/SNR、共模扰动、ADC 通道数、功耗或延迟对比。

### P1：非公度/低离散结构容差触觉界面

核心主张：

> HCP 微结构与梯度/非周期叉指电极降低装配偏移、旋转和接触半径变化导致的响应离散性。

重点关键词：

- low-dispersion tactile interface
- assembly tolerance
- gradient interdigitated electrodes
- non-periodic electrodes
- HCP/hexagonal microstructures
- response dispersion
- shift/rotation/contact-radius tolerance

### P2：模拟矢量触觉单元与 ADC 前方向解耦

核心主张：

> 四象限压阻单元通过模拟求和/差分，在 ADC 前输出 Kz/Kx/Ky，实现方向相关触觉预处理。

重点关键词：

- analog tactile preprocessing
- vector tactile sensing
- before digitization / pre-ADC
- summation and differencing circuit
- friction-aware sensing
- shear/friction cues
- direction-aware tactile readout

注意：如果没有纯三轴力标定矩阵，不应写 full 3D force sensor；应写 direction-related 或 vector-like tactile features。

### P3：低冗余矢量触觉阵列

核心主张：

> 2x2 单元作为 hardware vector macro-pixel，用 Kz/Kx/Ky 代替逐像素 raw readout，实现边缘、形状、接触图案或滑移方向感知。

重点关键词：

- hardware vector macro-pixel
- low-redundancy tactile array
- edge perception
- tactile array readout
- spatial tactile feature
- readout count reduction
- latency and power reduction

### P4：可编程物理触觉投影前端

核心主张：

> 3x3 压阻阵列通过可编程权重电压和电流求和电路，在 ADC 前直接生成 Ksum/Kx/Ky/Klap/Kring/Kcorner 等空间投影特征。

一句话：

> 不是采集后学习可迁移特征，而是在 ADC 前用电路把可迁移触觉特征物理算出来。

重点关键词：

- programmable physical tactile projection
- tactile front-end computing
- in-sensor computing
- spatial projection features
- current summation circuit
- single ADC tactile readout
- fault-tolerant tactile sensing
- transferable force perception
- cross-sensor calibration

## 材料体系与器件结构

已出现或可作为候选的体系：

- PDMS / Ecoflex / TPU 等柔性基底或微结构。
- HCP/Hex 微结构、棱台、泡沫、榫卯、鞋垫结构。
- PEDOT:PSS/EG、碳黑、CNT、MXene、AgNW、液态金属等导电材料需按实际数据筛选。
- PI/Cu、柔性 PCB、叉指电极、梯度叉指电极、非周期电极。
- 四象限压阻单元、2x2 macro-pixel、3x3 压阻阵列、4x4 阵列。
- 模拟求和/差分电路、可编程权重电压、电流求和、TIA、MUX、单 ADC 读出。

## 当前瓶颈

- 高配 Paper A 需要硬件 Kz/Kx/Ky 的 ADC 前证据，否则只能降级为后处理 vector features。
- 需要更系统的方向标定、PSD/SNR、共模抑制、功耗/延迟/ADC 通道数证据。
- P1 需要 shift、rotation、contact-radius、signal void ratio 等容差证据，而不是只证明灵敏度高。
- P3/P4 需要从单器件推进到阵列层面的边缘/形状/容错/迁移指标。
- 机器人闭环和滑移防护应用风险高，应作为最小演示，不应拖累 P1-P3 主线。

## 每日文献筛选优先级

每日自动文献任务应优先筛选以下论文：

1. ADC 前、模拟前端、in-sensor computing、sensor-near computing 相关触觉/电子皮肤论文。
2. 矢量触觉、剪切力/摩擦/滑移方向、三维力或 direction-aware tactile sensing。
3. 柔性触觉阵列的低通道读出、压缩读出、macro-pixel、edge/shape perception。
4. 可编程物理计算、投影核、模拟矩阵/电流求和、单 ADC 读出。
5. 电子皮肤容错、坏点鲁棒、跨器件迁移、少样本校准、domain adaptation。
6. 低离散结构、装配容差、微结构/电极结构协同设计。
7. 面向机械臂、鞋垫、可穿戴、空间人体监测的触觉/柔性电极系统，但必须能服务前端触觉计算主线。

## 不希望重复的方向

- 不要把用户方向简化成普通 flexible pressure sensor。
- 不要优先推荐只提高灵敏度、缺少结构机制或系统价值的材料论文。
- 没有标定证据时，不要把方向相关读出夸大成完整三维力重建。
- 不要把后端机器学习分类论文当成核心，除非它能和 ADC 前/传感器近端计算形成对照或启发。
- 不要让复杂机器人闭环应用压过器件、前端、阵列主线。

## 可迁移的创新方向

- 将 2x2 四象限矢量单元扩展为 3x3 可编程物理投影前端。
- 用硬件投影核生成 Ksum/Kx/Ky/Klap/Kring/Kcorner 等可解释空间特征。
- 用 normalized projection 特征降低跨器件差异和老化/坏点影响。
- 建立 hardware projection vs software projection 的一致性与失效边界。
- 做 raw pixel scanning、scalar downsampling、software gradient、hardware vector macro-pixel 的系统对照。
- 将摩擦/纹理/滑移方向线索作为 ADC 前矢量特征的任务价值，而不是只追求力数值重建。
